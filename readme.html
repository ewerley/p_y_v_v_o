Features

  • Optimization of Active Losses or Voltage-deviation in unbalaced distribution systems.

  • Handling of voltage, power factor and kVA constraints using quadratic penalty factors.

  • Integration with the OpenDSS format file.

  • Several control types to model capacitors, transformers and distributed generation.

  • Own PSO implementation with continuous and discrete variables handling, with or not velocity clamp and position clamp.

  • Several types of reports for improved optimization analysis.

Install

To install the latest version of PyVVO via command prompt or shell, type:

pip install -U git+https://github.com/yelrew/pyvvo

To install directly from the sourcs:

git clone https://github.com/yelrew/pyvvo.git && cd pyvvo && pip install -e .

Quick example

Active losses minimization on the IEEE13-bus feeder using PSO:

>>> import pyvvo
>>> vvo = pyvvo.Vvo(vvofile=pyvvo.resources_folder + '/IEEE13g.vvo')

>>> # Perform the optimization
>>> vvo.optimize(method='PSO', npart=10, itermax=15, discretization='end',
                 reports='info, base, solver, opt')

For extended usage, see the Tutorial section.
